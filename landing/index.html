Installation Steps

1

Start Ollama

ollama serve

Keep this running in a separate terminal

2

Pull AI Models

ollama pull deepseek-r1:8b
ollama pull llava:7b

Total size: ~12.5GB, runs in parallel, takes 10-30 min depending on internet

3

Install BrowserBot

pip install browserbot-mcp

Quick and clean. Installs Playwright and dependencies automatically

4

Configure Claude Desktop

Edit your config file at:

~/.claude/claude_desktop_config.json

Add this to the mcpServers section:

{
  "browserbot": {
    "command": "browserbot-mcp",
    "args": ["--headless"]
  }
}
5

Restart & Test

Quit and reopen Claude Desktop. In Claude, type:

"Open browser and navigate to example.com"

BrowserBot will take it from here

What BrowserBot Cannot Do

Brutal honesty about limitations. No BS.

CAPTCHA Solving

Google reCAPTCHA v3 blocks automated solving. You need manual completion or third-party services. We can't bypass that.

Mobile-Only Sites

Some apps detect desktop browser and block access. Playwright has limited mobile emulation. Complex apps may fail.

Rate Limiting

Sites with aggressive rate limiting will block you. We add delays but can't defeat IP bans or account throttling.

GPU Cost

Needs 24GB VRAM minimum. RTX 4090 territory. Not for budget setups. Monthly cloud GPU costs will exist.

We're Upfront

BrowserBot is powerful but not magical. We use Playwright (a real browser), which means you inherit all the limitations of web automation. Sites with advanced bot detection will still block you. We're fighting within the rules of the internet, not breaking them.
